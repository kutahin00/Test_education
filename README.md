# Программа защиты персональных данных в школе

Программа для безопасного хранения и обработки персональных данных учащихся, учителей и родителей в образовательных учреждениях.

## Описание

Программа разработана для дипломной работы "Защита персональных данных в школе" и предназначена для обеспечения конфиденциальности и безопасности персональных данных в образовательных учреждениях.

## Основные возможности

- ✅ Шифрование персональных данных с использованием алгоритма AES-256
- ✅ Безопасное хранение данных в зашифрованном виде
- ✅ Управление базой данных зашифрованных записей
- ✅ Валидация вводимых данных
- ✅ Экспорт и импорт данных
- ✅ Удобный графический интерфейс
- ✅ Поддержка различных типов данных (ученики, учителя, родители)
- ✅ **Интеграция с мессенджером MAX для отправки кодов подтверждения**
- ✅ **Двухфакторная аутентификация через коды подтверждения**

## Технологии

- **Python 3.x** - язык программирования
- **tkinter** - графический интерфейс (входит в стандартную библиотеку Python)
- **cryptography** - библиотека для криптографических операций
- **requests** - библиотека для работы с HTTP запросами (мессенджер MAX)
- **Алгоритм шифрования**: AES-256 (через Fernet)
- **Формат ключа**: PBKDF2 с SHA-256
- **Мессенджер MAX** - для отправки кодов подтверждения

## Использование

### 1. Настройка пароля

1. Откройте вкладку "Настройка пароля"
2. Введите пароль (рекомендуется не менее 8 символов)
3. Подтвердите пароль
4. Нажмите "Установить пароль"

**Важно**: Пароль необходим для шифрования и дешифрования данных. Запомните его или сохраните в безопасном месте!

### 2. Шифрование данных

1. Перейдите на вкладку "Шифрование данных"
2. Выберите тип данных (ученик, учитель, родитель)
3. Заполните поля с персональными данными
4. Нажмите "Зашифровать и сохранить"

Данные будут автоматически зашифрованы и сохранены в базу данных.

### 3. Дешифрование данных

1. Перейдите на вкладку "Дешифрование данных"
2. Введите ID записи из базы данных и нажмите "Загрузить и расшифровать"
   ИЛИ
   Вставьте зашифрованные данные в текстовое поле и нажмите "Расшифровать"

### 4. Настройка мессенджера MAX

1. Перейдите на вкладку "Мессенджер MAX"
2. Введите API ключ для доступа к мессенджеру
3. Укажите ID чата или номер телефона получателя
4. Нажмите "Сохранить настройки"
5. Проверьте подключение кнопкой "Тест подключения"

**Примечание**: Мессенджер используется для отправки кодов подтверждения при шифровании и дешифровании данных. Если API недоступен, сообщения сохраняются в файл `max_messenger_log.json`.

### 5. Управление базой данных

1. Перейдите на вкладку "База данных"
2. Просмотрите список всех записей
3. Используйте кнопки для управления записями:
   - "Обновить список" - обновить отображение записей
   - "Удалить запись" - удалить выбранную запись
   - "Экспорт в файл" - сохранить базу данных в файл

## Безопасность

- Все данные шифруются перед сохранением с использованием алгоритма AES-256
- Пароль используется для генерации ключа шифрования через PBKDF2
- Ключ шифрования никогда не хранится в открытом виде
- Используются современные криптографические стандарты
- **Двухфакторная аутентификация через коды подтверждения в мессенджере MAX**
- Коды подтверждения имеют ограниченное время жизни (10 минут)
- Все операции шифрования/дешифрования логируются и отправляются уведомления
- Данные валидируются перед шифрованием

## Формат данных

### Данные ученика (обязательные поля):
- Фамилия
- Имя
- Отчество
- Дата рождения (формат: ДД.ММ.ГГГГ)
- Класс

### Данные учителя (обязательные поля):
- Фамилия
- Имя
- Отчество
- Должность

2. Заполните поля:
   - Фамилия: Иванов
