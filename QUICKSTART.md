# Быстрый старт

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите программу:**
   ```bash
   python main_gui.py
   ```

## Первые шаги

1. **Установите пароль:**
   - Откройте вкладку "Настройка пароля"
   - Введите пароль (минимум 8 символов)
   - Подтвердите пароль
   - Нажмите "Установить пароль"

2. **Зашифруйте данные:**
   - Перейдите на вкладку "Шифрование данных"
   - Выберите тип данных (ученик/учитель/родитель)
   - Заполните поля
   - Нажмите "Зашифровать и сохранить"

3. **Просмотрите базу данных:**
   - Откройте вкладку "База данных"
   - Увидите все сохраненные записи

4. **Расшифруйте данные:**
   - Перейдите на вкладку "Дешифрование данных"
   - Введите ID записи или вставьте зашифрованные данные
   - Нажмите "Расшифровать"

## Важно!

⚠️ **Не забывайте пароль!** Без пароля невозможно восстановить данные.

## Пример данных

В файле `example_data.json` есть пример данных ученика, который можно загрузить через кнопку "Загрузить из файла".
